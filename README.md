## Global Scripts

- Build all

  ```sh
  npm run build
  ```

- Start all

  ```sh
  npm start
  ```

- Start docker all

  ```sh
  npm run docker:start
  ```

- Stop docker all

  ```sh
  npm run docker:stop
  ```

- Lint all

  ```sh
  npm run lint
  ```

- Test all

  ```sh
  npm test
  ```
